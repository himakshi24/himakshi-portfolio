# Himakshi Portfolio

Personal portfolio website — [Himakshi Rajesh Manmode](https://github.com/himakshi24).

## Live

- **Repo**: [github.com/himakshi24/himakshi-portfolio](https://github.com/himakshi24/himakshi-portfolio)

## Run locally

```sh
git clone https://github.com/himakshi24/himakshi-portfolio.git
cd himakshi-portfolio
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080).

## Build

```sh
npm run build
```

## Tech

- Vite · TypeScript · React
- shadcn/ui · Tailwind CSS
- React Router · Framer Motion
